# 2026 新年烟花 🎆

基于 Canvas API 实现的"2026"主题烟花动画效果。

## 功能特性

- ✅ **"2026" 艺术字体** - 手写风格，平滑淡入 + 缩放动画
  - 使用手写字体（Brush Script MT 等）
  - 橙红渐变：金黄→橙→橙红→深粉
  - 从小到大平滑放大（0.5x → 1.0x）
  - 从透明到完全不透明（3秒缓动动画）
  - 多重发光效果（橙色光晕 + 黄色高光）

- ✅ **5种绚烂烟花形态**
  - **球形绽放**（sphere）- 经典球形爆炸
  - **星形放射**（star）- 五角星图案，闪烁点缀
  - **柱状喷射**（column） - 垂直向上喷泉效果
  - **爱心形**（heart） - 浪漫心形图案（粉色）
  - **螺旋形**（spiral） - 彩虹螺旋效果

- ✅ **高级粒子系统**
  - 重力、阻力、轨迹拖尾（8个点）
  - 30% 粒子闪烁效果
  - 20% 粒子彩虹色变化
  - 每个粒子独立发光
  - 150 个粒子/烟花（更密集）

- ✅ **丰富配色方案**
  - 暖色系：红、橙、黄、粉
  - 冷色系：蓝、绿、紫
  - 闪烁色：白、柠檬、金
  - 多色混合爆炸效果

- ✅ **发光效果** - 使用 `globalCompositeOperation = "lighter"`

- ✅ **交互功能** - 鼠标点击触发、空格键自动模式（5%触发频率）

## 使用方法

### 1. 在浏览器中打开

直接双击 `index.html` 文件，或在浏览器中打开该文件。

### 2. 交互说明

- **点击屏幕** - 在点击位置触发随机类型的烟花
- **按空格键** - 开启/关闭自动燃放模式

## 技术实现

### 核心技术

- **Canvas API** - 所有图形渲染
- **ES6 JavaScript** - 面向对象的粒子系统
- **requestAnimationFrame** - 流畅动画循环

### 关键算法

1. **粒子物理系统**
   - 重力模拟
   - 空气阻力
   - 速度衰减

2. **烟花爆炸形态**
   - 球形：均匀向四周扩散
   - 星形：五角星放射状
   - 柱状：垂直向上喷射

3. **发光效果**
   ```javascript
   ctx.globalCompositeOperation = 'lighter';
   ```
   这个设置让重叠的粒子产生发光效果（模拟真实光线）

4. **轨迹拖尾**
   - 使用半透明背景覆盖：`rgba(0, 0, 0, 0.15)`
   - 保存粒子历史位置绘制连线
